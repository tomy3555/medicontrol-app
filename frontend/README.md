# Frontend - MediControl

Aplicación frontend desarrollada con React para la gestión de medicamentos, horarios y métricas de salud.

---

## Requisitos

- Node.js
- Yarn

---

## Instalación

1. Clonar el repositorio y cd frontend
2. Clonar el archivo .env.template y renombrarlo a .env
3. Ejecutar el comando `npm install` para instalar las dependencias
4. Verificar que el backend este corriendo en el puerto 3000
5. Ejecutar el comando `npm run dev` para iniciar el servidor de desarrollo
6. Ir a la URL desplegada en consola

Ejemplo:

```
http://localhost:5173/
```

## Notas

- La aplicación está desarrollada con React y TypeScript
- Se utiliza TailwindCSS y shadcn/ui para la interfaz
- La comunicación con el backend se realiza mediante Axios
- Requiere autenticación para acceder a la mayoría de las funcionalidades
